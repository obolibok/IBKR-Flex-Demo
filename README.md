# IBKR-Flex-Demo
Demo project for processing data from IBKR to Power BI via Flex Reports

## Prerequisites
*   **Python 3.8 or higher.**
*   **Microsoft Power BI Desktop.**
*   **IBKR Flex Query Reports (XML format):** You will need to create Flex reports for regular update and history import.
    1.  **Trades:** Orders executions.
    2.  **Cash Transactions:** Dividends, interest, withholding tax, fees, etc.
    3.  **Positions:** Positions history.
    4.  **Net Asset Value (NAV) in Base:** NAV history in base currency.
    5.  **Corporate Actions:** Details of splits, mergers, stock dividends, etc.
